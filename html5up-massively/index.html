Recipe Book

A place to compile all your screenshots of recipes into an online recipe book. I made use of SQL databses to store recipes as they were added to the site. I also made use of text extraction techniques using regex to identify the ingredients and instructions from a given screenshot. This project was coded in HTML/CSS, python, and javascript and connected through Flask Django.

Othello

This is the class game Othello in which you can choose to play against another player or an AI player. The UI is created using Tkinter, a standard GUI package for python. The AI algorithm used is the minimax algorithm which is used for decisions making in game theory. This project was coded entirely in python.

Coin Detection and Classification

This takes an inline parameter of an image and return an image with circles around each coin with different colors representing different coins (ie. red is penny, purple is quarter). This was made using the computer vision algorithms of Canny Edge Detection, Bresenham's, and Hough Transform. This project was coded entirely in C++.

Instagram Fashion Analyzer

A web app created to identify similar articles of clothing from a given instagram image. This project makes use of an instagram scraper to retrieve images from an account, a semantic segmentation model trained with the DeepFashion2 dataset, and Serp API to web scrape google results. This project was coded in HTML/CSS, javascript, and python.

Spotify Stats

This is essentially a constantly updating Spotify Wrapped. It allows you to see your top 10 most streamed songs on spotify in the past month. You can also search for songs and recieve information on the artist and album. This utilized concepts such as API's and OAuth. This project was coded in HTML/CSS and PHP.

Movie Recommendation

This is a basic movie recommendation system created using content based learning. This was made using the IMDB dataset along with the sklearn library to compute the similarity between attributes of the movies. This project was coded entirely in python.
